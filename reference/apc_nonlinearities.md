# Extract non-linear APC effects

Extract non-linear APC effects

## Usage

``` r
apc_nonlinearities(apc, intercept = FALSE)
```

## Arguments

- apc:

  APC model object from
  [`apc()`](https://elbersb.github.io/socialchange/reference/apc.md)

- intercept:

  Logical; include intercept in output

## Value

List of non-linear effect estimates
