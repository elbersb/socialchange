# Plot total APC effects under assumption

Plot total APC effects under assumption

## Usage

``` r
apc_plot_total(model, assumption)
```

## Arguments

- model:

  APC model object from
  [`apc()`](https://elbersb.github.io/socialchange/reference/apc.md)

- assumption:

  Character specifying linear trend assumption

## Value

ggplot2 object showing total effects
